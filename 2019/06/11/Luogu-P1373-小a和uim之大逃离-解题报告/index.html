P1373 小a和uim之大逃离

题意简述

感觉题目描述挺清楚的

给定矩阵，两个人以任意点为起点，每次可以向下走或向右走，同时两人的得分交替加上当前格的值并对$k + 1$取模，可以在任意点由后者结束。求两人得分一致的方法数。

解题思想

题目中的方案数与两人的具体得分无关，故我们考虑将每一步两人的差值作为状态。剩下的状态显而易见：当前位置和取数人。即$dp[x][y][\Delta][op]$表示在坐标为$(x, y)$的方格上由$op$取数，取完后两人的差值为$\Delta$

因为由后者结束，所以我们只统计$op = 0$时的答案，并累加。再加上题目中给出的移动方式，我们不难得出状态转移方程：

$$dp[x][y][\Delta][0] = dp[x - 1][y][\Delta - a[x][y]][1] + dp[x][y - 1][\Delta - a[x][y]][1]$$

$$dp[x][y][\Delta][1] = dp[x - 1][y][\Delta + a[x][y]][0] + dp[x][y - 1][\Delta + a[x][y]][0]$$

需要考虑差值小于0的情况，这时将插值替代为模$k + 1$意义下的等价元就行了。
